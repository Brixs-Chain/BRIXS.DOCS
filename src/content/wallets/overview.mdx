## Integration Pathways


### Connecting to MetaMask (Brixs Testnet)

You can easily add the Brixs Testnet to your MetaMask wallet to interact with dApps and send $BRIXS. 

To add the network manually, open MetaMask, go to **Add Network > Add a network manually**, and use the following details:

- **Network Name:** Brixs Testnet
- **New RPC URL:** `https://rpc-testnet.brixs.space`
- **Chain ID:** `7777777`
- **Currency Symbol:** `BRIXS`
- **Block Explorer URL:** `https://testnet.brixs.space`

<Callout type="info" title="Testnet Faucet">
Need testnet tokens? Head over to the [Brixs Faucet](https://faucet.brixs.space/) to claim testnet $BRIXS and start building.
</Callout>
