## Core Ecosystem Pillars

### 1. The Sovereign Blockchain Node (Testnet)
Our custom EVM node is built from the ground up to support our L2 architecture. It is currently configured to Chain ID `7777777` with the native gas token `BRIXS`.

```bash
# Connect your local environment to the Brixs Testnet
const provider = new ethers.providers.JsonRpcProvider("http://localhost:8545");
```

### 2. The Zero-Gas Wallet (Account Abstraction)
Through native EIP-4337 integration, the Brixs Wallet acts as a smart contract. When a user clicks "Swap", our custom RPC intercepts the transaction, and the Paymaster handles the gas fees transparently in the background.

### 3. The Developer CLI
We are building the ultimate command-line tool for developers. Deploying a smart contract will be as simple as a single command, instantly pushing your logic to the chain.

```bash
# Deploy instantly to the Testnet
brixs deploy
```
